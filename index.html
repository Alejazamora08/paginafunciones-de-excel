Funciones de Excel

Las funciones en Excel son herramientas poderosas que te permiten realizar cálculos y análisis de datos de manera eficiente

Funciones más usadas

Nombre Descripción Uso Formula
Suma La función SUMA se utiliza para sumar un rango específico de celdas Puedes emplear esta función para agregar rápidamente un conjunto de números en una columna o fila. =SUMA(A1:A5)
Si La función SI realiza una evaluación condicional y devuelve un valor según el resultado de la evaluación Se utiliza para realizar decisiones basadas en condiciones. Por ejemplo, puedes utilizar la función SI para mostrar "Aprobado" si la puntuación es mayor o igual a 70 y "Reprobado" si es inferior a 70 =SI(A1>=70, "Aprobado", "Reprobado")
